## trond snapshot source

Show available snapshot source.

### Synopsis

Support different types of snapshot (Fullnode, Lite Fullnode), in different regions (Singapore, America).


```
trond snapshot source [flags]
```

### Examples

```
# Show available snapshot source
$ ./trond snapshot source

```

### Options

```
  -h, --help   help for source
```
